## About

This repository contains practical examples, exercises, and small projects that help reinforce the basics of JavaScript. It is intended for beginners or anyone looking to strengthen their JS skills.

## Topics Covered

- Variables and Data Types
- Operators and Expressions
- Functions
- Control Flow (`if`, `switch`, loops)
- Arrays and Objects
- DOM Manipulation
- Event Handling
- Basic ES6 Features (let, const, arrow functions, template literals)
